🧭 Анти ИИ Навигатор (Прямо) Голосовой навигатор помогает идти прямо. Он помогает человеку идти в заданном направлении и не сбиться с пути.
## 🎯 Как это работает

1. Навели телефон в нужную сторону → нажали кнопку
2. Убрали телефон в карман → пошли
3. Если отклоняетесь влево → голос говорит **«Левее»**
4. Если отклоняетесь вправо → голос говорит **«Правее»**
5. Если идёте назад → голос говорит **«Разворот! Вы идете назад»**

---

## 🌲 Для кого

- Для леса — когда нет троп и ориентиров
- Для темноты — когда ничего не видно
- Для экстренных ситуаций — когда нужно быстро выйти к людям
- Для охоты, рыбалки, грибов, туризма

---

## 🔧 Особенности

- 🔊 **Голосовые подсказки** — не нужно смотреть на экран
- 📱 **Работает в кармане** — руки свободны
- 🔋 **Экономит батарею** — чёрный экран + тихий пульс
- 🛡️ **14 систем защиты** — не собьётся, не зависнет
- 🌐 **Полностью офлайн** — без интернета

---

## 📱 Установка

1. Откройте `index.html` в Chrome на телефоне
2. Нажмите «⋮» → «Установить приложение»
3. Иконка появится на рабочем столе
